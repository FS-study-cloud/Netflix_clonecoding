# 넷플릭스 클론코딩 리포트

## 개요

넷플릭스 사이트를 클론코딩하여 React, HTML, CSS에 대해 복습 그리고 심화 학습을 위해 진행되었다.

## 기대효과

- React 기반 웹 개발
- Typescript 활용
- 도커(Docker)를 활용한 배포
- AWS 환경 이해 및 활용

[역할](%EC%97%AD%ED%95%A0%202c0c9cce983880ecbd16de78173db1a3.csv)

## 기능적 요구사항

1. 검색창
    1. 검색 창에 콘텐츠를 입력하면 연관 콘텐츠가 출력된다.
    2. 만약 해당 콘텐츠가 없을 시 알려주는 창이 출력된다.
2. 미디어쿼리/반응형 화면
    1. 화면의 크기에 따라 보여지는 콘텐츠 수가 달라진다.
3. 모달
    1. 콘텐츠를 클릭했을 때 해당 콘텐츠의 예고편이나, 사진이 뜨도록 한다.
